Wesnoth Campaign Tools Suite

A collection of powerful utilities for Battle for Wesnoth campaign developers

Streamline your Wesnoth campaign development with this suite of specialized tools. Extract, analyze, and manage your scenario files with ease, allowing you to focus on creating immersive stories and engaging gameplay.

All tools are designed specifically for Wesnoth's WML format to ensure compatibility and preserve your scenario structure.

Scenario Event & Message Extractor

Extract and manage events and messages from your scenario files. Edit dialogues, review event structures, and export modified files.

  • Extract all events and their messages
  • Edit messages directly in the interface
  • Preserve WML structure during editing
  • Export modified scenario files
  • Multi-file batch processing

Scenario Story Extractor

Extract and organize story content, character dialogues, and scenario information from your campaign files.

  • Extract story text from [story] tags
  • Organize dialogues by character and event
  • Format text with proper sentence breaks
  • Identify speaker information
  • Export story content for editing

Unit Balance Analyzer

Analyze and compare unit statistics to ensure balanced gameplay across factions and scenarios.

  • Compare unit stats side-by-side
  • Calculate combat effectiveness metrics
  • Identify overpowered/underpowered units
  • Faction balance analysis
  • Cost-effectiveness calculations

Interactive Map Viewer

Visualize and edit your scenario maps with an interactive editor that works directly with Wesnoth map format.

  • Import and export Wesnoth map files
  • Real-time terrain editing
  • Starting position management
  • Village and castle placement tools
  • Scenario testing preview

Campaign Validator

Validate your campaign files for common errors and WML syntax issues before testing in-game.

  • WML syntax validation
  • Dependency checking
  • Missing file detection
  • Performance analysis
  • Compatibility checks

Dialogue Tree Generator

Create and visualize branching dialogue trees for your campaign events with an intuitive interface.

  • Visual dialogue tree editor
  • Character-specific dialogues
  • Condition and effect management
  • Export to WML event format
  • Dialogue flowchart visualization
