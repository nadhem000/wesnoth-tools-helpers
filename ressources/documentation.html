Introduction

Welcome to the Wesnoth Campaign Tools Suite documentation. This comprehensive guide will help you effectively use our tools to create and manage Battle for Wesnoth campaigns.

The suite includes three main tools: Unit Editor for creating custom units, Scenario Builder for designing campaign scenarios, and WML Validator for ensuring your code is error-free.

Key Features

  • Real-time preview of units and scenarios
  • Drag-and-drop interface for map creation
  • Advanced WML validation with error highlighting
  • Multi-language support for international campaigns
  • Export functionality for Wesnoth-compatible files

Unit Editor

Create and customize units with detailed statistics and abilities for your campaign.

Creating Units

  1. Click 'New Unit' from the dashboard
  2. Set base attributes: HP, movement, cost
  3. Add attacks with type, damage, and specials
  4. Define resistances and vulnerabilities
  5. Add abilities and traits
  6. Preview and save unit
Note: All units must have a unique ID. Use the 'Validate' button to check for compatibility issues.

Scenario Builder

Design and build engaging scenarios with terrain, units, and events for your campaign.

Terrain Placement

Use the terrain palette to paint maps. Right-click to sample terrain, left-click to paint.

Unit Placement

Drag units from the sidebar onto the map. Set ownership, starting positions, and AI behavior.

Events Manager

The Events Manager tool allows you to extract, edit, and manage events and messages from Wesnoth scenario files.

Using the Events Manager

  1. Upload scenario files (.cfg) using the drag-and-drop zone or the browse button
  2. Paste WML content directly into the text area and click 'Process Pasted Content'
  3. Switch between Events View and Messages View using the toggle buttons
  4. Edit event code and message text directly in the editors
  5. Save changes and download the edited files

Key Features

  • Extract events and messages from multiple scenario files
  • Group content by file or view combined results
  • Real-time statistics on events and messages
  • Direct editing of WML event code
  • Message translation and editing interface
Note: The Events Manager preserves your original files and only modifies content when you explicitly save changes.

Troubleshooting

If events aren't showing up, ensure your files contain valid [event] tags. The parser looks for these tags to identify events.

Story Manager

The Story Manager tool helps you create and manage narrative sequences, dialogues, and branching story paths for your campaigns.

Using the Story Manager

  1. Create a new story or open an existing campaign
  2. Add characters with portraits and dialogue styles
  3. Build dialogue trees with player choices and consequences
  4. Connect story segments to scenario events
  5. Preview the narrative flow and character interactions
  6. Export to WML for use in your campaign

Key Features

  • Visual dialogue tree editor with drag-and-drop interface
  • Character library with portrait management
  • Branching narrative paths with conditional logic
  • Integration with Events Manager for in-game triggers
  • Export to WML [story] format

Export Example

[story]
    [part]
        story= _ "Once upon a time in the Kingdom of Wesnoth..."
    [/part]
    [part]
        story= _ "A young hero appears at the castle gates"
        image=portraits/hero.png
    [/part]
[/story]
Note: The Story Manager maintains character consistency across your campaign scenarios. Use the "Link to Event" feature to connect dialogue choices with game events.

Best Practices

  • Use meaningful IDs for all elements
  • Comment complex WML sections
  • Test scenarios with different difficulty levels
  • Balance unit statistics carefully
  • Use the validator before each export

Frequently Asked Questions

Can I import existing WML files?

Yes! Use the 'Import' option from the File menu. The tool will parse and convert existing WML into editable objects.

How do I contribute to the project?

Visit our GitHub repository to report issues, suggest features, or submit pull requests.

Are there any sample campaigns?

Yes, under 'Templates' you'll find several sample campaigns demonstrating various features.
