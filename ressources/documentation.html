Introduction

Welcome to the Wesnoth Campaign Tools Suite documentation. This comprehensive guide will help you effectively use our tools to create and manage Battle for Wesnoth campaigns.

The suite includes many inactive tools: Unit Editor for creating custom units, Scenario Builder for designing campaign scenarios, and WML Validator for ensuring your code is error-free.

Key Features

  • Real-time preview of units and scenarios
  • Drag-and-drop interface for map creation
  • Advanced WML validation with error highlighting
  • Multi-language support for international campaigns
  • Export functionality for Wesnoth-compatible files

Unit Editor

Create and customize units with detailed statistics and abilities for your campaign.

Creating Units

  1. Click 'New Unit' from the dashboard
  2. Set base attributes: HP, movement, cost
  3. Add attacks with type, damage, and specials
  4. Define resistances and vulnerabilities
  5. Add abilities and traits
  6. Preview and save unit
Note: All units must have a unique ID. Use the 'Validate' button to check for compatibility issues.

Scenario Builder

Design and build engaging scenarios with terrain, units, and events for your campaign.

Terrain Placement

Use the terrain palette to paint maps. Right-click to sample terrain, left-click to paint.

Unit Placement

Drag units from the sidebar onto the map. Set ownership, starting positions, and AI behavior.

Events Manager

The Events Manager tool allows you to extract, edit, and manage events and messages from Wesnoth scenario files.

Using the Events Manager

  1. Upload scenario files (.cfg) using the drag-and-drop zone or the browse button
  2. Paste WML content directly into the text area and click 'Process Pasted Content'
  3. Switch between Events View and Messages View using the toggle buttons
  4. Edit event code and message text directly in the editors
  5. Save changes and download the edited files

Key Features

  • Extract events and messages from multiple scenario files
  • Group content by file or view combined results
  • Real-time statistics on events and messages
  • Direct editing of WML event code
  • Message translation and editing interface
Note: The Events Manager preserves your original files and only modifies content when you explicitly save changes.

Troubleshooting

If events aren't showing up, ensure your files contain valid [event] tags. The parser looks for these tags to identify events.

Story Manager

The Story Manager tool extracts and organizes story content, including dialogues and narrative parts, from Wesnoth scenario files.

Using the Story Manager

  1. Upload scenario files (.cfg) using the file upload or paste the content directly
  2. The tool will automatically extract story parts and dialogues
  3. View the extracted content organized by scenario and events
  4. Export the story content to WML format for reuse

Key Features

  • Extract story parts and dialogues from scenario files
  • View scenario metadata including ID and next scenario
  • Organize content by events with message dialogues
  • Export to WML [story] format

Export Example

[story]
						[part]
						story= _ "Once upon a time in the Kingdom of Wesnoth..."
						[/part]
						[part]
						story= _ "A young hero appears at the castle gates"
						image=portraits/hero.png
						[/part]
					[/story]
Note: The Story Manager extracts narrative content from your scenario files. Use the export feature to create standalone story files.

GIF Creator

The GIF Creator tool allows you to create animated GIFs from image sequences entirely in your browser.

Using the GIF Creator

  1. Select images using the file picker (JPG, PNG, etc.)
  2. Adjust frame delay to control animation speed (default 200ms)
  3. Click "Create GIF" to generate your animation
  4. Preview your GIF - it will appear in the result section
  5. Right-click on the GIF and choose "Save image as..." to download

Key Features

  • Client-side processing - no images uploaded to servers
  • Adjustable frame delay for animation speed control
  • Image previews before GIF creation
  • Real-time processing status indicators
Note: All processing happens locally in your browser - no images are uploaded to any server!

Big Map Tracker

The Big Map Coordinate Tracker tool allows you to track coordinates on your images, place markers, and export the data. All processing happens locally in your browser.

Using the Big Map Tracker

  1. Upload an image using the 'Upload Image' button
  2. Hover over the image to see coordinates
  3. Click to place markers of the selected type
  4. Use the command interface to run multiple placement commands
  5. Save your project or export the image with markers

Key Features

  • Local processing - no images uploaded to servers
  • Multiple marker types (dot, cross, circle, target, square)
  • Command interface for batch operations
  • Project saving/loading using browser storage
  • Export image with markers overlay
  • Full undo/redo history
Note: All your data remains in your browser. You can download the project to save it permanently.

Unit Comparator

The Unit Comparator tool allows you to upload multiple unit configuration files, compare their stats, and calculate various effectiveness metrics.

Using the Unit Comparator

  1. Upload unit configuration files (.cfg) using the drag-and-drop area or the browse button
  2. Optionally, load sample units to see how the tool works
  3. Select a comparison metric from the available options
  4. Adjust sorting and filtering options as needed
  5. Click 'Compare Units' to generate the comparison table
  6. Click on any unit row to view detailed information

Key Features

  • Compare multiple units side-by-side
  • Multiple metrics for unit effectiveness
  • Sort by any attribute or calculated metric
  • Filter units by race/faction
  • Robust parser with default values for missing attributes
  • Sample units for demonstration
  • Detailed log of processing and warnings
Note: The parser will automatically fill in missing attributes with default values and provide warnings about any issues found in the unit files.

.po File Dictionary Manager

The .po File Dictionary Manager helps you manage translation dictionaries for Wesnoth campaigns. It extracts terms from .po files, creates translation dictionaries, and applies them to other .po files.

Using the Dictionary Manager

  1. Upload .po files or paste content directly
  2. Select your target language from the dropdown
  3. Choose processing options (single words, two words, or both)
  4. Click 'Process Files' to extract dictionary entries
  5. Edit, save, or download the dictionary as needed
  6. Use the dictionary to apply translations to other .po files

Key Features

  • Extract terms from .po files to build translation dictionaries
  • Filter by word count (single words or two-word phrases)
  • Edit dictionary entries directly in the interface
  • Save dictionaries to browser storage for future use
  • Apply dictionaries to modify .po files with translations
  • Export modified .po files with applied translations
  • All processing happens locally in your browser

Workflow Example

1. Upload English-to-French .po file
						2. Process to create French dictionary
						3. Upload untranslated .po file
						4. Apply French dictionary
					5. Download modified .po file with translations
Note: The Dictionary Manager preserves your original .po files and only modifies content when you explicitly save changes.

PO File Manager

The PO File Manager provides a comprehensive interface for managing PO translation files. It allows you to extract strings, manage translations, and generate translated PO files while preserving all metadata and formatting.

Using the PO Manager

  1. Upload .po files or paste content directly into the source textarea
  2. Select target language from the dropdown menu
  3. Click 'Extract Strings' to parse the PO file and display translatable content
  4. Translate strings directly in the table cells or bulk upload translations
  5. Use 'Apply Translations' to generate the translated PO file
  6. Download the translated PO file with language code in filename

Key Features

  • Full PO file parsing with metadata extraction
  • Interactive translation table with editable cells
  • Bulk operations (Fuzzy All, Unfuzzy All, Empty Shell)
  • Plural form handling with contextual explanations
  • Multiple export formats (CSV, with context, filtered sets)
  • Translation statistics and analysis reports
  • All processing happens locally in your browser

Workflow Example

1. Upload original .po file
2. Extract strings and translate messages
3. Apply translations to generate output
4. Download es.po (Spanish translation)
5. Use Analyze to compare original and translated files
Note: The PO Manager preserves all comments, references, and metadata during processing. Obsolete messages (#~) are maintained but excluded from translation counts.

Best Practices

Frequently Asked Questions

Can I import existing WML files?

Yes! Use the 'Import' option from the File menu. The tool will parse and convert existing WML into editable objects.

How do I contribute to the project?

Visit our GitHub repository to report issues, suggest features, or submit pull requests.

Are there any sample campaigns?

Yes, under 'Templates' you'll find several sample campaigns demonstrating various features.
